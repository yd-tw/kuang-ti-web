## 專案介紹

這個專案的啟發是來自於期望優化網站而生。過往我都是使用Google協作平台來創建網站，但由於樣式固定且功能較少，因此我自行上網學習如何撰寫現代化的響應是網站。這個專案是我參考「Judy Gab」所發佈的教學製作而成的，MIT授權協議我放在third_license中。
